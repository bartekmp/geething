Messages

When enabled, messages you mark as read stay visible in the list (dimmed) and are removed on the next refresh or when the popup is reopened. By default they disappear immediately.

Groups related Gmail™ messages into collapsible threads in the message list. Turn off to show each message as a separate item.

Privacy

When enabled, remote images in email bodies (including tracking pixels) are blocked. Disabled by default — images load normally.

Keyboard Shortcuts

Active in the popup when focus is not on a button or input.

j / ↓ Next message or thread
k / ↑ Previous message or thread
Enter Open message · expand or collapse thread
Escape Close detail / close popup
r Mark focused message or thread as read
a Archive focused message or thread
o Open focused message or thread in Gmail™
